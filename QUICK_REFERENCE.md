## 🆘 If There's a Problem

### App won't load?
- Clear browser cache
- Refresh page
- Check Vercel dashboard status

### PWA won't install?
- Must be HTTPS (Vercel has this)
- Try different browser
- Check manifest.json is served

### Need to make changes?
- Edit code locally
- Push to GitHub
- Vercel auto-deploys
