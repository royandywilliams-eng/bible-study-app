# PWA (Progressive Web App) - Installation Guide

## What is a PWA?

A Progressive Web App (PWA) is a web application that works like a native mobile app. The Bible Study App is now a fully functional PWA that:

✅ **Installs on your home screen** - Just like a native app
✅ **Works offline** - All data syncs locally
✅ **Fast loading** - Cached for instant access
✅ **Responsive design** - Optimized for all devices
✅ **No app store needed** - Install directly from the web

---

## How to Install

### On Android 📱

1. **Open the app in Chrome**
   - Visit: https://bible-study-app.example.com
   - Make sure you're using Chrome, Edge, or Brave

2. **Click the install button**
   - Look for the **"Install"** button in the browser address bar
   - Or tap the **menu (⋮)** → **"Install app"**

3. **Confirm installation**
   - Tap **"Install"**
   - The app appears on your home screen
   - Open from home screen icon

### On iPhone/iPad 📱

1. **Open the app in Safari**
   - Visit: https://bible-study-app.example.com
   - Use Safari (other browsers won't support installation)

2. **Add to home screen**
   - Tap the **Share** button (box with arrow)
   - Select **"Add to Home Screen"**

3. **Confirm and launch**
   - Tap **"Add"**
   - Icon appears on your home screen
   - Opens like a native app

### On Desktop 💻

1. **Open in Chrome/Edge/Brave**
   - Visit: https://bible-study-app.example.com

2. **Click install button**
   - Look for **"Install"** button in address bar
   - Or use **menu (⋮)** → **"Install [App Name]"**

3. **Launch from applications**
   - Opens in standalone window (no browser UI)
   - Access from applications menu

## Managing Offline Data

### View Storage Usage
1. **On Android**: Settings → Apps → Bible Study → Storage
2. **On iPhone**: Settings → Safari → App Data
3. **On Desktop**: App settings → Storage

### Clear Cache (if needed)
1. **On Android**: Settings → Apps → Bible Study → Clear Cache
2. **On iPhone**: Settings → Safari → App Data → Remove All
3. **On Desktop**: Windows: App settings → Clear cache

### Export Your Data
1. Open the app
2. Go to Settings
3. Click "Export Data"
4. Save the JSON file to your device
5. Keep for backup or import to another device

## Troubleshooting

### App won't install
- **Check browser**: Use Chrome, Edge, Brave, or Safari
- **Check HTTPS**: App must be served over HTTPS (https://, not http://)
- **Check history**: Clear browser cache and try again
- **Try again later**: Some browsers need multiple visits before showing install prompt

### Offline content not loading
- **First load**: Requires internet to cache content
- **Clear cache**: Go to app settings and clear cache
- **Reinstall**: Uninstall and reinstall the app
- **Update**: Make sure you have the latest version

### Data not syncing
- **Check storage**: Device may be running low on space
- **Clear cache**: Remove old cached data
- **Restart**: Close and reopen the app

### App won't update
- **Check internet**: Must be connected to download updates
- **Clear cache**: Try clearing app cache in settings
- **Reinstall**: Uninstall and reinstall for latest version

## Performance

### Typical Usage
- **First load**: 30-60 seconds (downloads and caches app)
- **Bible load**: 10-20 seconds first time (caches 2-3 MB)
- **Subsequent loads**: < 1 second (cached)

### Storage Usage
- **App shell**: 5 MB
- **Bible data**: 2-3 MB (compresses from 10 MB)
- **Your notes/progress**: < 1 MB (unless you have many notes)
- **Total typical usage**: 10-15 MB
